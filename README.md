# Food Rating App Frontend

## Overview
The Food Rating App is a frontend application for users to rate and review various food items. This application provides an intuitive interface that allows users to manage and display their ratings efficiently.

## Features
- **User Authentication**: Users can sign up and log in to the application.
- **Rating System**: Users can rate food items on a scale of 1 to 5.
- **Review Submission**: Users can write reviews along with their ratings.
- **Food Item Display**: The app showcases various food items with their current ratings and reviews.
- **Responsive Design**: The application is designed to work on both desktop and mobile devices.

## Technologies Used
- **React**: The frontend is built with React for a dynamic user experience.
- **Redux**: State management is handled using Redux.
- **CSS**: Styling is done using CSS for a clean and modern design.
- **Axios**: For API calls to the backend server.

## Installation
To run the Food Rating App locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/rateit-frontend.git
   ```
2. Navigate into the project directory:
   ```bash
   cd rateit-frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
Once the application is running, you can:
- Sign up or log in to your account.
- Browse food items, rate and review them.
